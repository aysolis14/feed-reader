# Feed-Reader Project
This project involves writing test suites with Jasmine for a web application that reads RSS feed.

## Test Functions
To begin, first download the zip folder to run locally on your computer, or clone the repository. Click the green "Clone or Download" button. All files inside the respository are REQUIRED. 

- The menu 
    - the element is hidden by default
    - visibility is changed when clicked
- Initial Entries
    - has at least a single .entry element
- New Feed Selection
    - content is changed
- RSS Feeds
    - URL is defined and not empty
    - name is defined and not empty
    - are defined

## Dependencies 
- Jasmine 3.1.0
- Google Fonts Roboto
